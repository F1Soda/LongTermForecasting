# Долгосрочное прогнозирование акций

Логическое продолжение [предыдущего доклада](https://github.com/F1Soda/MPT-report). 

Этот проект посвящён долгосрочному прогнозированию цен акций на основе исторических данных с использованием базовых 
инструментов теории вероятности и экономики

Работа основана на статье Magnus Erik Hvass Pedersen](https://github.com/Hvass-Labs) —
[Long-Term Stock Forecasting](https://raw.githubusercontent.com/Hvass-Labs/Finance-Papers/master/pedersen2020long-term_stock_forecasting.pdf)
и представляет собой её практический анализ и адаптацию для широкого круга читателей

Основная цель — показать на реальных примерах, как работает модель долгосрочного предсказания, описанная в оригинальной 
статье, и дать понятный, воспроизводимый инструмент для экспериментов. Никаких больших кусков текстов и кода. Всё
идёт последовательно, объединяя 
[теорию](https://raw.githubusercontent.com/Hvass-Labs/Finance-Papers/master/pedersen2020long-term_stock_forecasting.pdf) 
с [практикой](https://github.com/Hvass-Labs/FinanceOps/blob/master/Paper_Long-Term_Stock_Forecasting.ipynb) 
оригинала, скрывая детали реализации в отдельные файлы c подробной документацией

В репозитории вы можете найти исходный код, python ноутбук и его pdf версию
